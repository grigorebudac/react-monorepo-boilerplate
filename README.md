# React monorepo boilerplate

## Features

- lerna
- typescript
- @redux/toolkit
- react-router-dom
- storybook
- @testing-library/react
- eslint & prettier

## Development setup

```sh
yarn install
yarn lerna:bootstrap
yarn start
```

## Generating templates

The boilerplate is able to generate code following a predefined pattern.

### Try it out

```sh
yarn generate
```
